##
### Criação do projeto
##

- Instalar o Expo
- Expo-Cli
```
npm install -g expo-cli
```

- Expo Version
```
expo --version
```

- Git Version
```
git --version
```

- Instalar o Android Studio para fazer a emulação do dispositivo mobile
- Configurar ...
- `More Action/Virtual device menager/Create virtual device/`

**Criando o projeto com Expo**
- Projeto Expo
```
npx expo init myapp
```
